## Project Overview
SignalFriend is a Web3 signal marketplace on BNB Chain where:
- Predictors sell trading signals as NFTs
- Traders buy signals and receive transferable receipt NFTs
- All payments in USDT, transparent on-chain
- Ratings are entirely off-chain in MongoDB

## Tech Stack
| Component | Technologies |
|-----------|-------------|
| Smart Contracts | Solidity, Foundry, OpenZeppelin (BNB Chain) |
| Backend | Express.js, MongoDB, Viem, SIWE, JWT, Zod, Pino |
| Frontend | React, Viem, Wagmi, RainbowKit (planned) |

## Important Patterns
- Smart contracts use 3-of-3 MultiSig governance
- Two-phase deployment (script + manual MultiSig setup)
- Backend verifies NFT ownership + MongoDB records
- SIWE for wallet auth, JWT for session management
- Alchemy webhooks for blockchain event indexing
- Feature-based folder structure in backend
- Ratings are ENTIRELY off-chain in MongoDB
